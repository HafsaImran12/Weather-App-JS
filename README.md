# 🌦️ Weather App

A modern and responsive Weather App built using **HTML, CSS, Bootstrap, and JavaScript**. This application allows users to search for any city and instantly view real-time weather information using the OpenWeather API.

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Current temperature
- ☁️ Weather condition
- 💧 Humidity
- 💨 Wind speed
- 📅 Current day & date
- 🌍 Country name
- 🎨 Dynamic weather icons
- 📱 Fully responsive design
- ⚠️ Error handling for invalid city names

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- OpenWeather API

## 🌐 API Used

**OpenWeather API**

Provides real-time weather data for cities around the world.
